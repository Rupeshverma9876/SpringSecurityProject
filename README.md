This is a simple login authentication project built using Spring boot and Spring Security. It allows the  users to login securely with username and password validation.

Features :-
User login with Spring Security,
Username and password authentication,
Access control for protected pages

Technologies Used :- 
Java,
Spring Boot,
Spring Security,
Maven,
HTML/CSS,
Thymeleaf (optional),
IntelliJ IDEA / Eclipse

Working of Spring Security Project :-
In these Project first user  have to register then the detail of user will shown on userdashboard  and then after they access the particular pages by using login functionalities with username and password which are created in registeration time. the profile page will be secure using spring security and these page can only access after login with correct username and password .

NOTE:-
These project is focused on backend functionality, especially Spring Security configuration and user authentication.  
UI/Frontend is simple and UI is not the main focus.
